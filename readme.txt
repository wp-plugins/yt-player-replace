 Replace Youttube Player with VideoJS HTML5 player

== Description ==

This plugin allows you to easily replace Youtube player with VideoJS

Additional features of the plugin controlled both via settings and filters:

* Install ** Video.js - HTML5 Video Player for WordPress** 
* change colors for your player 
* change size for your player in all posts


== installation==


From your WordPress dashboard

   1)  Visit 'Plugins > Add New'
   2)  Search for 'bbPress'
   3)  Activate Youtube Player replace from your Plugins page. 
   4) Done now all automaticaly change youtube Videos with VideoJs
